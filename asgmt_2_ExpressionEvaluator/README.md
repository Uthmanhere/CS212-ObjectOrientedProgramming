# The Expression Evaluator
Evaluates the expression through a very friendly 
and readable C++ Object Oriented design. 

## Exit
Press the *EOF* key :
	C+c
to exit.

## Installation
On commandline, run :
	make
to make *build* directory containing the executable.
Run the program using :
	./build/cal
